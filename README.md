# Customer Churn Analysis

## Project Overview
This project analyzes customer churn patterns and revenue trends using Python, SQL, and Power BI.

The goal is to understand customer behavior, identify churn drivers, and generate business insights.

## Tools Used
Python (Pandas)
SQL
Power BI

## Dataset
Customer transaction and demographic dataset.

## Analysis Workflow

1 Data Cleaning using Python  
2 Exploratory Data Analysis  
3 SQL queries for business insights  
4 Power BI dashboard visualization  

## Key Insights

• Certain marketing channels generate higher revenue  
• Premium loyalty customers churn less frequently  
• Some products contribute most of the total revenue
