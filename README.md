# Customer Churn Analysis | Python, SQL, Power BI

## Project Overview
Customer churn is one of the biggest challenges for businesses because losing customers directly impacts revenue.  

This project analyzes customer churn patterns using Python, SQL, and Power BI to identify key factors influencing customer retention and revenue generation.

The goal is to provide data-driven insights that help businesses understand customer behavior and reduce churn.

---

## Objectives
- Analyze customer churn patterns
- Identify high-value customer segments
- Understand revenue contribution by product
- Analyze marketing channels and their impact on revenue
- Build an interactive dashboard for business insights

---

## Tools & Technologies
- Python (Pandas)
- SQL
- Power BI
- GitHub

---

## Dataset
The dataset contains customer demographic and transactional information such as:

- Customer segment
- Product purchased
- Revenue
- Marketing channel
- Loyalty tier
- Payment method
- Age group
- Churn status

This data is used to analyze customer behavior and revenue trends.

---

## Project Workflow

### 1 Data Cleaning
Data was cleaned and transformed using Python to ensure accuracy and consistency.

### 2 Exploratory Data Analysis
Python was used to explore patterns in customer churn and revenue.

### 3 SQL Analysis
SQL queries were written to generate business insights such as revenue contribution and customer segmentation.

### 4 Dashboard Visualization
An interactive dashboard was built using Power BI to visualize key metrics and insights.

---

## Key Business Insights
- Certain marketing channels generate significantly higher revenue.
- Some customer segments show a higher churn rate.
- Premium loyalty tier customers tend to churn less.
- A small number of products contribute to the majority of revenue.
